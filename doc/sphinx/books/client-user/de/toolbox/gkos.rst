GeKoS
=====

Das |gkos| ``GeKoS``-Werkzeug. GeKos ist ein Anbieter für kommunale Software Lösungen. Falls Ihre Kommune mit GeKoS zusammenarbeitet, können Sie ,mit Hilfe von diesem Werkzeug, Daten und Informationen direkt weiterleiten.


Für ausführliche Informationen schauen Sie gerne auch unter: https://www.gekos.de/

Verlinkungen
============

Die GBD WebSuite ermöglicht das Einbinden von individuellen Icons in der Statusleiste am unteren Bildschirmrand.
Es können sämtliche SVG-Dateien als Icons verwendet werden. Über diese Icons können dann verschiedene Arten von Verlinkungen definiert werden.
Eine Weiterleitung zu einer Website, welche sich dann in Ihrem Internetbrowser öffnet, wäre möglich, aber auch eine Verlinkung zu anderen Elementen in der GBD WebSuite.
So könnten zum Beispiel Informationen zum Datenschutz elegant integriert werden.
Das Erstellen und Einbinden dieser Verlinkungen ist im GBD Admin Handbuch beschrieben.
